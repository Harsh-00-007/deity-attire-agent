An end-to-end multi-agent workflow that autonomously generates culturally specific attire (Hindu deity dresses/poshaks) and intricate card designs using a custom-trained Stable Diffusion LoRA.

## 🚀 Overview

This project bridges the gap between Large Language Models and fine-tuned Vision Models. Instead of manually prompting an image generator, a designated **Creative Director Agent** interprets high-level design requirements, constructs highly specific visual prompts, and orchestrates the generation process through a custom fine-tuned LoRA model.

## 🏗️ System Architecture

1. **User Input:** User provides a high-level theme (e.g., "Design a festive Diwali collection").
2. **LLM Agent (LangChain):** Parses the theme, brainstorms concepts, and generates optimized positive and negative prompts.
3. **Inference Pipeline (HuggingFace Diffusers):** 
   - Base Model: `Stable-Diffusion-v1.5` (or SDXL)
   - Custom Weights: `poshak_card_lora_v1.safetensors`
4. **Output:** A generated portfolio of cohesive designs.

## 📊 Dataset & Fine-Tuning (LoRA)

Training an accurate model for specialized cultural attire required a custom dataset pipeline:
* **Collection:** Curated 300+ high-resolution images of traditional card designs and deity poshaks.
* **Preprocessing:** Center-cropped to 512x512, normalized, and captioned using BLIP-2 for accurate text-image pairs.
* **Training:** Trained using `kohya_ss` scripts.
  * Rank (dim): 128
  * Learning Rate: 1e-4
  * Epochs: 15
